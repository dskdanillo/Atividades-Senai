Estrutura de Repetição “Para Faça” (For)

A estrutura de repetição for é um tipo de laço de controle utilizado em programação para executar um bloco de código um número determinado de vezes, geralmente com base num contador ou indexador. Essa estrutura é ideal para situações onde a quantidade de repetições é conhecida de antemão, pois o for combina a inicialização de uma variável de controle, a condição de continuação do laço e a atualização (incremento ou decremento) do contador em uma única linha de código, garantindo uma execução eficiente e limpa.

SINTAXE BÁSICA
for(expressao) {
  for(inicialização; condição; incremento)
}     // comandos que serão executados

Exemplo:

  for(let i = 1; i <= 5; i++) {
    console.log("Repetição número: " + i);
}
