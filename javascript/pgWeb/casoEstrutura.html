Estrutura condicional “caso” (switch / case)

O switch é usado quando você quer testar uma variável ou expressão contra vários valores possíveis. É como se fosse “vários if encadeados”, mas mais organizado.

INICIO

SINTAXE BÁSICA
switch(expressao) {
  case valor1:
    // Código executado se expressao === valor1
    break;
  case valor2:
    // Código executado se expressao === valor2
    break;
  default:
    // Código executado se nenhum dos casos for verdadeiro
}
