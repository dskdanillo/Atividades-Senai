Estrutura de Repetição Faça enquanto

É uma estrutura de repetição (loop) que executa um bloco de código pelo menos uma vez e continua executando enquanto uma condição for verdadeira.

INICIO

SINTAXE BÁSICA
do { // Código a ser executado } while (condição);
